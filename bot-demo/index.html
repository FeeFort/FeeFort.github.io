The bot already supports commands such as !ask, !balance, !casino, !roulette, !color, and !duel.

Casino, roulette, betting, and economy features are purely for entertainment purposes. The bot currency comes from Twitch channel points and exists only as an internal channel balance. It has no real-world value, monetization, withdrawal, exchange, or purchase functionality.

The !whois Command

The !whois command was developed to make high-level VALORANT leaderboard statistics more accessible and entertaining for Twitch chat.

Viewers can enter a leaderboard position and retrieve statistics from the previous VALORANT Act, including recent results, K/D ratio, and headshot percentage.

The Riot API integration for this feature is implemented in cogs/whois.py. Some TODO comments remain because development API access does not fully expose all production response scenarios yet, making certain edge cases impossible to completely verify at this stage.
🏆

Leaderboard Lookup

Retrieves a player from the previous VALORANT Act leaderboard using leaderboard position.

/val/ranked/v1/leaderboards/by-act/{actId}
📦

Act Detection

Detects the currently active Act and automatically identifies the previous one.

/val/content/v1/contents
📜

Recent Match History

Loads recent matches for the selected player using their Riot PUUID.

/val/match/v1/matchlists/by-puuid/{puuid}
📊

Detailed Match Statistics

Calculates K/D ratio, headshot percentage, and recent results from detailed match data.

/val/match/v1/matches/{matchId}
⚠️

TODO Verification

Remaining TODO items include verification of production response formats, match history ordering, and draw handling behavior.

🔒

Error Handling

Handles invalid leaderboard positions, hidden player profiles, empty histories, and unavailable players.
