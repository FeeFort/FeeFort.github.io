# Команды бота

## !help

### Использование

```
!help
```
Отправляет сообщение со списком всех доступных команд.

---

## !ask

### Использование

```
!ask [вопрос]
```

Позволяет задать вопрос ИИ.

---

## !balance

### Использование

```
!balance
```

Показывает текущий баланс пользователя. Монеты покупаются за баллы канала.

---

## !pay

### Использование

```
!pay [@тег] [кол-во]
```

Позволяет перевести монеты другому участнику чата.

## !roulette

### Использование

```
!roulette [ставка]
```

«Русская рулетка».

- Коэффициент: **x1.5**
- Шанс победы: **1 к 6** (~**16,7%**)

---

## !duel

### Использование

```
!duel [@имя_участника] [ставка]
```

### Принять или отклонить дуэль
```
!duel accept/decline
```

Предлагает дуэль другому участнику чата.

- Победитель забирает ставку в двойном размере
- Проигравший теряет свою ставку
- Шансы победы: **50/50**

---

## !color

### Использование

```
!color [ставка] [цвет EN]
```

Цветная рулетка. Нужно угадать цвет.

- Если угадываешь — получаешь **x1.5** от своей ставки
- Шанс победы при выборе конкретного цвета: **1 к 9** (~**11,1%**)
